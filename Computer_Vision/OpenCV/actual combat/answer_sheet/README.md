# 答题卡判题
基于OpenCV实现答题卡正确率的检测。

运行环境：
- Python 3.9.21
- opencv-python 4.11.0.86

主要内容：
- get_answer 检测答题卡正确率

运行方式：python get_answer.py -i images/example_test.png
