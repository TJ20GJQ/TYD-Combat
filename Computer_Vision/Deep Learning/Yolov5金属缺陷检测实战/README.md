# Yolov5金属缺陷检测实战
YOLOv5 是一种基于深度学习的目标检测算法，属于 YOLO（You Only Look Once）系列模型的第五代版本。它以高效、准确和易用性著称，广泛应用于实时目标检测任务。

运行环境：
- Python 3.9.21
- CUDA 12.6
- torch 2.6.0+cu124
- torchvision 0.21.0+cu124

主要内容：
- train 基于Yolov5网络预训练模型进行钢材表面缺陷数据集NEU-DET的训练
- detect 基于训练的Yolov5模型进行钢材表面缺陷检测
