# 新闻数据集文本分类实战

运行环境：
- python 3.9.21
- CUDA 12.6
- torch 2.6.0+cu124
- tenserboardX 2.5.0
- scikit-learn 1.6.1

主要内容：
- text_classification_run 使用TextRNN/TextCNN模型进行新闻数据集文本分类

  运行命令：python .\text_classification_run.py --model TextRNN（TextCNN）
