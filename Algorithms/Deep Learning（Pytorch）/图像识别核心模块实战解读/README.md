# 图像识别核心模块实战解读

运行环境：
- Python 3.9.21
- CUDA 12.6
- torch 2.6.0+cu124
- torchvision 0.21.0+cu124
- scikit-learn 1.6.1

主要内容：
- CNN_mnist 训练CNN网络进行Mnist分类任务
- flower_classify 训练Resnet网络进行花卉分类任务
- flower_classify_test 加载训练模型，使用测试集进行花卉测试
